# Smart Hospital Resource Allocation System

A full-stack healthcare management platform designed to optimize hospital resource allocation using AI. Built with Flask, SQLite, and Google Cloud Run, this system leverages Google Gemma to provide predictive analytics and intelligent resource management.

## 🚀 Features

- **AI-Powered Resource Allocation**: Utilizes Google Gemma to predict demand and optimize bed/doctor allocation.
- **Real-Time Dashboard**: Monitor hospital capacity, waiting times, and resource usage in real-time.
- **Smart Patient Triage**: AI chatbot and intake system to prioritize critical cases.
- **Doctor & Staff Management**: Efficient scheduling and shift management tools.
- **Responsive Design**: Modern, continuous scrolling layout accessible on all devices.

## 🛠️ Tech Stack

- **Backend**: Python, Flask, SQLAlchemy
- **Frontend**: HTML5, CSS3, JavaScript (Chart.js for analytics)
- **Database**: SQLite (Development), PostgreSQL (Production ready)
- **AI/ML**: Google Gemma, Vertex AI
- **Deployment**: Google Cloud Run, Docker

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/smart-hospital-system.git
   ```
2. Install dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```
3. Run the application:
   ```bash
   python run.py
   ```
