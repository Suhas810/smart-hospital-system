## 📌 About the Project

Smart Hospital System is a full-stack web application developed as my **first major learning project** in web development.

The goal of this project was to understand how real-world applications are built by integrating the frontend, backend, database, and deployment into a complete system.

The application provides essential hospital management functionalities like patient records, doctor management, appointment scheduling, authentication, and administrative operations.

While the project is functional, it was primarily built for learning purposes and serves as a strong foundation for future improvements.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Suhas810/smart-hospital-system.git
```

## Navigate into the Project

```bash
cd smart-hospital-system
```

## Install Dependencies

```bash
pip install -r backend/requirements.txt
```

## Run the Application

```bash
python run.py
```

The application will start locally.
